## Character Encoding
When the file is opened you can set the encoding which will be used when saving the file. You can access the encoding menu by clicking CTRL + P, on Windows, or CMD + P, on MAC, and entering "$Encoding". Here's the list of all supported encoding:

* ASCII
* Big5
* EUC-KR
* GB2312
* KOI8-R
* CP1256
* ISO-8859-1
* ISO-8859-2
* ISO-8859-3
* ISO-8859-4
* ISO-8859-5
* ISO-8859-6
* ISO-8859-7
* ISO-8859-8
* ISO-8859-9
* ISO-8859-13
* ISO-8859-14
* ISO-8859-15
* JIS
* Windows-1250
* Windows-1251
* Windows-1252
* Windows-1253
* Windows-1254
* Windows-1255
* Windows-1256
* Windows-1257
* Windows-1258
* UTF-7
* UTF-8
* UTF-16
* UTF-32
